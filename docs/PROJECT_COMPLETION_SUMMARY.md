## 📋 快速部署指南

### 1. 环境准备
```bash
# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件，添加必要的配置
```

### 2. 数据库设置
```bash
# 推送数据库结构
supabase db push

# 运行迁移脚本
npm run migrate
```

### 3. 部署
```bash
# 开发模式
npm run dev

# 生产部署
npm run deploy
```

### 4. Telegram 设置
```bash
# 设置 Webhook
curl -X POST https://your-domain.com/api/telegram/webhook/set

# 设置菜单
./scripts/setup-telegram-menu.sh
```
