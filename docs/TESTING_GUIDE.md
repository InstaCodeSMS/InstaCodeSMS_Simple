## 🚀 快速测试步骤

### 第1步：环境准备
```bash
# 1. 检查依赖
npm install

# 2. 检查环境变量
cat .env

# 3. 检查数据库连接
npm run dev
```

### 第2步：测试 Telegram Bot
```bash
# 1. 启动开发服务器
npm run dev

# 2. 在 Telegram 中测试
# 打开 Telegram，搜索您的 Bot
# 测试以下命令：
/start          # 启动 Bot
/help           # 查看帮助
/receive        # 开始接码
/orders         # 查看订单
/products       # 查看产品
```

### 第3步：测试 Mini App
```bash
# 1. 访问 Mini App
# 在浏览器中打开：https://your-domain.com/mini-app

# 2. 测试功能
# - 浏览产品列表
# - 添加到购物车
# - 结算流程
# - 订单查询
```
